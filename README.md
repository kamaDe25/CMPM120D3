# CMPM120D3

Gameplay/Experience Requirements
The game uses continuous input by tracking how long the player takes to complete each level.
The game uses discrete inputs such as buttons progress through level and click to pick up items.

Player's goal can be achieved indirectly
    In all levels player must gather items before reaching the stove.
    In level 1 player must turn off fan to gather the rest of the items.
    In level 2, Player must push block to keep button turned on. This will turn on the fan and 
        allow them to get the last item.
    In level 3, player must push a specific item down slide without, falling down the slide themselves. 

Game has 3 levels.
    Each level has a new obstacle for the player to overcome.

Other scenes used to contextualize the game
    Start scene and level summary scenes
