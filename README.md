# CMPM120D3

Gameplay/Experience Requirements
The game uses continuous input such as the arrow keys.
The game uses discrete inputs such as buttons that turn on fans and platforms.

Player's goal can be achieved indirectly
    In all levels player must make sure cheese is caught in bowl. 

Game has 3 levels.
    Each level has a new challenge for the player.

Other scenes used to contextualize the game
    Studio Intro scene, game title scene, end scene
